# RateGuard Node SDK

RateGuard's Node SDK is a middleware package for Express, Fastify, Hono, and Next.js route handlers.

It is separate from `packages/sdk-ts`, which is the generated control-plane client.

## Install

```bash
npm install @rateguard/node
```

or

```bash
bun add @rateguard/node
```

## Status

This package is being added as the Node middleware counterpart to `packages/sdk-go`.
